Release Notes

V1.1.0 (Current)

  • Added 21 text effects: CFadeA03, CStepA01-A04, CCustomA01-A05, TFadeA03, TStepA01-A05, TCustomA01-A05.

  • Added 13 color modes: ColorBurn, ColorDodge, Darken, Divide, Exclusion, HardLight, Lighten, LinearLight, Normal, Overlay, Screen, SoftLight, Subtract.

  • Added locked status for the Animatext component.

  • Now the execution order of the Animatext component is 0.

  • Fixed incorrect text animation when the Animatext component was enabled again.

  • Removed 4 text effects: CBasicA05, CElasticA05, CBounceA05, CWaveA05.

V1.0.2 (2021-12-03)

  • Added new integrations: PlayMaker, Quest Machine.

  • Fixed incorrect text during text update in the Animatext component.

  • Fixed incorrect effect state when the Animatext component is refreshed.

V1.0.1 (2021-11-05)

  • Added effect previews to the documentation.

  • Improved word parsing and line parsing in the "Textinfo" class.

  • Fixed incorrect text animation when front interval of execution effect is less than 0.
